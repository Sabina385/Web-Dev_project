# Movie Club

## Description
Movie Club is a web application built with Angular and Django REST Framework.  
Users can add movies, write reviews, rate films, and recommend movies to friends.

## Tech Stack
**Frontend:**
- Angular
- TypeScript

**Backend:**
- Django
- Django REST Framework (DRF)
